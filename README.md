# Finance calculator (React, Redux, TypeScript)

A budget planning app. Demo is available [here](https://fincalc-branxy.vercel.app/).

## Architecture

### UI

- [React](https://react.dev/)
- [Radix UI](https://www.radix-ui.com/)

### State Management

- [Redux Toolkit](https://redux-toolkit.js.org/)

### Database

- [Supabase](https://supabase.com/) (work in progress, currently only auth is added)

## Installation

1. Clone the repository (or download ZIP)

```
git clone https://github.com/branxy/finance-calculator-react-redux.git
```

2. Install dependencies:

```
cd finance-calculator-react-redux
npm i
```

3. Start the dev server:

```
npm run dev
```
